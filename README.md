## Abstract

汉字不同于其他表音性的文字，其表义的特殊性质为中文词语的分布式表示提供了另外的可能。其中，部件，作为具有组配汉字功能的构字单位，其具有的表义能力又成为汉语字、词级别的嵌入不可忽视的要素。以下文章主要从部件、字、词表示几个角度出发，到分词、词性标注等基础任务，再到文本分类、机器翻译等下游任务，以时间为序，梳理了汉字部件相关的向量表示及具体应用方面的最新研究成果。
